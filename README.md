# Watermark Adder - Digital Content Protection

A high-performance web tool for adding text or logo watermarks to images with precision and ease.

## 🌟 Features
- **Batch Processing**: Add watermarks to multiple images simultaneously.
- **Customization**: Adjust position, size, opacity, and font for text watermarks.
- **Logo Support**: Upload and scale your own branding logos.
- **Live Preview**: Real-time visual feedback of watermark placement.
- **Privacy**: Processing happens entirely in the browser; no images are uploaded to a server.

## 🛠️ Tech Stack
- **Frontend**: HTML5 Canvas API, CSS3, Vanilla JavaScript.

## 📁 Key Files
- `index.html`: Main application interface.
- `script.js`: Core logic for image processing and canvas manipulation.
- `style.css`: Visual styling for the tool.
